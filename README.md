# label-tool
A tool for labeling anomaly of multivariate time series.

## Getting Started

#### clone the repo

```shell
git clone https://github.com/CaoYuanpu/label-tool.git
```

#### Install dependencies (with python>=3.5)

```shell
cd label-tool
pip install -r requirements.txt
```

#### Run the code

```shell
cd ui
python config_page.py
```
